# N-Gram Text Generation

## Run the bot

Place the **json** with chats in the main folder and run:

```bash
python -m bot
```

## Configuration file

- Create a **config.ini** file

```ini
[TELEGRAM_CHAT]
ids = XXXXXXXX YYYYYYYY ZZZZZZZZ
```
